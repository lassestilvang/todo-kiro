# Product Overview

Daily Task Planner is a modern task management web application for organizing daily tasks, projects, and to-dos.

## Core Features

- **Task Management**: Create, edit, complete, and delete tasks with rich metadata (descriptions, dates, deadlines, priorities, estimated time)
- **Organization**: Group tasks into custom lists with emojis and colors
- **Labels**: Tag tasks with customizable labels (icons and colors)
- **Subtasks**: Hierarchical task structure with parent-child relationships
- **Smart Views**: Inbox, Today, Upcoming, Next 7 Days, All Tasks, and custom list/label views
- **Recurring Tasks**: Support for daily, weekly, monthly, yearly, and custom recurring patterns
- **Reminders**: Set multiple reminders per task
- **Attachments**: Upload and manage files attached to tasks
- **Change History**: Track all modifications to tasks
- **Search**: Full-text search across tasks using Fuse.js
- **Keyboard Shortcuts**: Comprehensive keyboard navigation and shortcuts

## User Experience

- Clean, professional interface with dark/light theme support
- Accessibility-first design (WCAG 2.1 AA compliant)
- Smooth animations with reduced motion support
- Responsive design for desktop and mobile
- Offline-capable with local SQLite database
