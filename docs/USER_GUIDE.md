## Getting Started

### First Launch

When you first open Daily Task Planner, you'll see:
- A sidebar on the left with navigation
- The main panel showing your Inbox (empty at first)
- A header with search and theme toggle

### Creating Your First Task

1. Click the **"Add Task"** button in the sidebar (or press `Cmd/Ctrl + K`)
2. Enter a task name (required)
3. Optionally add more details:
   - Description
   - Date (when you plan to work on it)
   - Deadline (when it must be done)
   - Priority (High, Medium, Low, or None)
   - Estimated time
4. Click **"Create Task"** or press Enter

Your task now appears in the Inbox!

## Task Management

### Creating Tasks

**Quick Add**:
- Click "Add Task" button
- Press `Cmd/Ctrl + K` (keyboard shortcut)
- Enter minimal information and save

**Detailed Add**:
- Use the task form to add all properties:
  - **Name**: What needs to be done (required)
  - **Description**: Additional context or notes
  - **Date**: When you plan to work on it
  - **Deadline**: When it must be completed
  - **Priority**: High (red), Medium (yellow), Low (blue), or None
  - **Estimated Time**: How long you think it will take (HH:mm format)
  - **Actual Time**: Track time spent (HH:mm format)
  - **Labels**: Tag with one or more labels
  - **Recurring Pattern**: Make it repeat automatically
  - **Reminders**: Set notification times

### Editing Tasks

**Quick Edit**:
- Click the checkbox to mark complete/incomplete
- Click the task name to open the detail view

**Full Edit**:
1. Click on a task to open the detail view
2. Modify any field
3. Changes are saved automatically
4. Close the detail view when done

### Completing Tasks

- Click the checkbox next to a task
- Completed tasks show with strikethrough text
- Toggle "Show Completed" to hide/show completed tasks
- Completion time is automatically recorded

### Deleting Tasks

1. Open the task detail view
2. Click the **"Delete Task"** button
3. Confirm the deletion
4. Task and all related data (subtasks, attachments, etc.) are removed

### Task Properties

**Priority Levels**:
- **High**: Red indicator, for urgent/important tasks
- **Medium**: Yellow indicator, for important tasks
- **Low**: Blue indicator, for less urgent tasks
- **None**: No indicator, default priority

**Time Tracking**:
- **Estimated Time**: Your guess for how long it will take
- **Actual Time**: Track real time spent
- Format: Hours and minutes (e.g., 2:30 for 2 hours 30 minutes)

**Dates**:
- **Date**: When you plan to work on the task (shows in Today/Upcoming views)
- **Deadline**: When the task must be done (used for overdue tracking)
- Both are optional

1. Open a task detail view

## Views

Views are different ways to filter and display your tasks.

### Inbox

- Shows all tasks in your Inbox list
- Default view for new tasks
- Good for quick capture and processing

### Today

- Shows tasks scheduled for today (date = today)
- Includes overdue tasks (deadline passed, not completed)
- Focus on what needs attention now

### Upcoming

- Shows all tasks scheduled for today or future dates
- Excludes tasks without dates
- Good for planning ahead

### Next 7 Days

- Shows tasks scheduled within the next week
- Helps you plan your week
- Excludes tasks beyond 7 days

### All

- Shows every task regardless of date or list
- Useful for reviewing everything
- Can be overwhelming for large task lists

### Custom List Views

- Click any list in the sidebar
- See only tasks in that list
- Organized by your custom categories

### Custom Label Views

- Click any label in the sidebar
- See all tasks with that label
- Cross-list view of related tasks

### Show/Hide Completed Tasks

- Toggle button in each view
- Hide completed tasks to reduce clutter
- Show completed tasks to review what you've done

## Search

The search feature helps you quickly find tasks by name or description.

### Using Search

1. Click the search box in the header (or press `Cmd/Ctrl + F`)
2. Start typing
3. Results appear in real-time
4. Matching text is highlighted

### Search Features

- **Fuzzy Matching**: Finds approximate matches (typos are okay)
- **Real-time Results**: Updates as you type
- **Highlights**: Matching text is highlighted in yellow
- **Fast**: Results appear within 300ms

### Search Tips

- Search works across task names and descriptions
- Shorter queries may return more results
- Use specific keywords for better results
- Search is case-insensitive

### Navigating Results

- Click any result to open that task
- Use arrow keys to navigate results
- Press Enter to open the selected result
- Press Escape to close search

## Recurring Tasks

Recurring tasks automatically repeat on a schedule, saving you from manually recreating the same task.

### Creating a Recurring Task

1. Create or edit a task
2. Expand the "Recurring Pattern" section
3. Choose a pattern:
   - **Every day**: Repeats daily
   - **Every week**: Repeats weekly on the same day
   - **Every weekday**: Monday through Friday only
   - **Every month**: Same date each month
   - **Every year**: Same date each year
   - **Custom**: Define your own pattern
4. Optionally set an end date
5. Save the task

### How Recurring Tasks Work

- When you complete a recurring task, a new instance is automatically created
- The new task has the same properties (name, description, priority, etc.)
- The date is calculated based on the pattern
- Original task is marked complete and archived

### Custom Patterns

For custom patterns, you can define:
- Specific days of the week
- Specific dates of the month
- Complex schedules

Example: "Every Monday and Thursday" or "1st and 15th of each month"

### Editing Recurring Tasks

**Edit One Instance**:
- Open the task and make changes
- Only that instance is affected

**Edit the Pattern**:
- Open the task
- Modify the recurring pattern
- Future instances will use the new pattern

### Stopping Recurrence

1. Open the recurring task
2. Remove the recurring pattern
3. Or set an end date in the past

## Subtasks

Break down complex tasks into smaller, manageable subtasks.

### Creating Subtasks

1. Open a task detail view
2. Scroll to the "Subtasks" section
3. Click **"Add Subtask"**
4. Enter the subtask name
5. Press Enter or click "Add"

### Managing Subtasks

**Complete a Subtask**:
- Click the checkbox next to the subtask
- Subtask is marked complete
- Parent task shows progress (e.g., "2/5 subtasks complete")

**Edit a Subtask**:
- Click on the subtask to open its detail view
- Subtasks are full tasks with all properties
- Can have their own dates, priorities, labels, etc.

**Delete a Subtask**:
- Open the subtask detail view
- Click "Delete Task"
- Subtask is removed from parent

### Subtask Display

- Parent tasks show subtask count (e.g., "3 subtasks")
- Completion status is shown (e.g., "2/3 complete")
- Subtasks appear nested under parent in task lists

### Best Practices

- Use subtasks for multi-step tasks
- Keep subtasks focused and actionable
- Complete subtasks in order for better tracking
- Don't nest subtasks more than one level deep

## Attachments

Attach files to tasks for reference, documentation, or context.

### Adding Attachments

**Drag and Drop**:
1. Open a task detail view
2. Drag a file into the attachment area
3. File uploads automatically

**File Picker**:
1. Open a task detail view
2. Click **"Add Attachment"**
3. Select a file from your computer
4. File uploads automatically

### Supported File Types

- Documents: PDF, DOC, DOCX, TXT, etc.
- Images: JPG, PNG, GIF, SVG, etc.
- Spreadsheets: XLS, XLSX, CSV, etc.
- Presentations: PPT, PPTX, etc.
- Archives: ZIP, RAR, etc.

**File Size Limit**: 10MB per file

### Managing Attachments

**View Attachments**:
- Open a task detail view
- Scroll to the "Attachments" section
- See list of all attached files with icons

**Download Attachments**:
- Click the download icon next to a file
- File downloads to your default download location

**Delete Attachments**:
- Click the delete icon next to a file
- Confirm deletion
- File is removed from task and filesystem

### Attachment Display

- File name is shown
- File size is displayed
- File type icon helps identify the file
- Upload date is recorded

## Reminders

Set reminders to get notified about tasks at specific times.

### Adding Reminders

1. Open a task detail view
2. Scroll to the "Reminders" section
3. Click **"Add Reminder"**
4. Select a date and time
5. Click "Add"

### Multiple Reminders

- Add multiple reminders per task
- Useful for:
  - Initial reminder (e.g., 1 day before)
  - Follow-up reminder (e.g., 1 hour before)
  - Final reminder (e.g., at deadline)

### Managing Reminders

**Edit a Reminder**:
- Click on the reminder
- Change the date/time
- Save changes

**Delete a Reminder**:
- Click the delete icon next to the reminder
- Reminder is removed

### Reminder Behavior

- Reminders trigger at the specified time
- Notification appears in the application
- Triggered reminders are marked as "triggered"
- Reminders don't repeat (use recurring tasks for that)

**Note**: Browser notifications require permission. Grant permission when prompted for best experience.

### Viewing Change History

## Tips and Best Practices

### Task Management

1. **Use the Inbox for Quick Capture**
   - Don't overthink it, just add tasks quickly
   - Process and organize later

2. **Set Realistic Deadlines**
   - Give yourself buffer time
   - Use deadlines for truly time-sensitive tasks

3. **Estimate Time Conservatively**
   - Tasks usually take longer than expected
   - Add 25-50% buffer to estimates

4. **Review Regularly**
   - Check Today view each morning
   - Review Upcoming view weekly
   - Process Inbox daily

5. **Complete Tasks Daily**
   - Build momentum by completing tasks
   - Start with quick wins
   - Tackle high-priority items first

### Organization

1. **Keep Lists Focused**
   - Don't create too many lists
   - 3-7 lists is usually enough
   - Use labels for additional categorization

2. **Use Labels Strategically**
   - Context-based labels (Home, Office, Computer)
   - Time-based labels (Quick, Long)
   - Theme-based labels (Important, Ideas)

3. **Break Down Large Tasks**
   - Use subtasks for multi-step tasks
   - Each subtask should be completable in one sitting
   - Aim for 3-7 subtasks per task

4. **Leverage Recurring Tasks**
   - Use for habits and routines
   - Weekly reviews, monthly reports, etc.
   - Saves time and ensures consistency

### Productivity

1. **Start with Today View**
   - Focus on what's scheduled for today
   - Don't get overwhelmed by everything

2. **Use Priority Wisely**
   - Not everything is high priority
   - Reserve high priority for truly urgent items
   - Most tasks should be medium or low

3. **Time Block Your Day**
   - Use estimated time to plan your day
   - Schedule tasks in your calendar
   - Leave buffer time between tasks

4. **Batch Similar Tasks**
   - Use labels to group similar tasks
   - Complete all "Calls" at once
   - Process all "Quick" tasks together

5. **Review and Reflect**
   - Check change history to see progress
   - Review completed tasks weekly
   - Adjust estimates based on actual time

### Maintenance

1. **Archive Completed Tasks**
   - Hide completed tasks to reduce clutter
   - Review completed tasks periodically
   - Delete old completed tasks if needed

2. **Clean Up Labels**
   - Remove unused labels
   - Consolidate similar labels
   - Keep label list manageable

3. **Organize Lists**
   - Delete empty or unused lists
   - Merge similar lists
   - Keep list structure simple

4. **Manage Attachments**
   - Delete unnecessary attachments
   - Keep file sizes reasonable
   - Use descriptive file names

## Troubleshooting

### Task Not Showing in View

- Check if the task has the correct date
- Verify the task is in the expected list
- Make sure "Show Completed" is enabled if the task is complete
- Try searching for the task

### Search Not Finding Task

- Check spelling (fuzzy search helps but isn't perfect)
- Try shorter or different keywords
- Verify the task exists in the database

### Attachment Upload Fails

- Check file size (max 10MB)
- Verify file type is supported
- Ensure you have disk space available
- Try a different file

### Keyboard Shortcut Not Working

- Make sure you're not in a text input field
- Check if another app is using the same shortcut
- Try clicking outside any input first

### Theme Not Saving

- Check browser local storage permissions
- Try clearing browser cache
- Manually set theme again
